# JapanZipcode

[![Gem Version](https://badge.fury.io/rb/japan_zipcode.svg)](https://badge.fury.io/rb/japan_zipcode)

## Usage

```ruby
gem 'japan_zipcode'
```

and run `bundle install`

Generate Migrations:

```bash
$ rake japan_zipcode:install:migrations
```

and run `rails db:migrate`

Fill data:

```bash
$ rake japan_zipcode_data
```
